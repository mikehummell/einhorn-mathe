## ✨ Features

- Keine Installation — direkt im Browser spielen
- Funktioniert auf Mobile & Desktop
- Gamification: Punkte, Streak, Level-Ups, Upgrades
- Turbo-Modus mit Zeitbonus
- Diplom als PDF herunterladen
- Schweizerdeutsche Wörter & Begriffe

---

## 🛠️ Technik

Reine HTML/CSS/JavaScript Dateien — kein Framework, kein Backend, kein Tracking.
Hosting via GitHub Pages.
